Badge Criteria

Criteria for all "brillianICM" modules - What it takes to earn this badge

Successful completion of the serious game brillianICM which includes the following activities:

  • Theory Module "Intercultural Management" which contains:

    • Fundamental classification of intercultural management
    • Classification of each country in the cultural world map
    • Guessing the 9 globe dimension values of each country
    • Explanation and execution of the four-sides model of Friedemann Schulz von Thun
    • Classification of the visible and invisible parts of the Ice-Berg Model
